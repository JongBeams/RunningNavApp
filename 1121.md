### **모바일 앱 (프론트엔드)**

**기본 UI**

- 홈 화면 UI [구현] 
- 러닝 홈 화면 UI [구현] 
- 코스 목록 화면 UI [구현] 
- 마이페이지 화면 UI [구현] 
- 마이페이지 -> 러닝 기록 
- 마이페이지 -> 설정

**인증 & 사용자**

- 회원가입( DB에 사용자 등록)
- 로그인/로그아웃
- 설정 (알림, 단위 설정)

**코스 생성 & 관리**

- 지도 표시 (Naver Map API) [구현]

- 시작/도착 지점 설정
- 거리/예상 시간 자동 계산
- 코스 저장 (제목, 설명, 난이도, 태그)
- 내 코스 목록 조회/수정/삭제
- 코스 미리보기

**네비게이션**

- 실시간 위치 추적 (GPS) [구현]
- 현재 위치 표시 [구현]
- 코스 안내 시작/일시정지/종료
- TTS 음성 안내 (방향, 거리, 코스 이탈 알림)
- 진행 상황 표시 (거리, 시간, 속도)

**활동 기록**

- 운동 기록 목록
- 통계 (주간/월간 거리, 시간)

**사용자 API**

- 프로필 조회/수정

- 내 코스 목록
- 코스 상세 정보
- 코스 경로 데이터 (GeoJSON)
- 거리/예상 시간 계산

**활동 기록 API**

- 운동 기록 생성/조회
- 통계 데이터

**관리자**

- 사용자 관리 (조회, 정지)
